# 智能曆 (SmartCalendar)

## 版本資訊
- **Version**: 1.0.0

## 核心功能
1. **任務與月曆管理**：提供直覺的月曆視圖與待辦事項清單，輕鬆規劃每日行程，並包含資料自動儲存功能。
2. **多功能生活小工具**：內建股票價格追蹤、當地天氣即時資訊、以及專注用的番茄鐘。
3. **磁吸側邊欄設計**：擴展功能以無邊框獨立視窗形式存在，拖曳主視窗時會自動磁吸對齊，不佔用多餘畫面空間。

## 系統需求
- **作業系統**：Windows 10 / Windows 11 (64-bit)
- 無須安裝 Python 等額外環境，解壓縮後即可直接執行。

## 如何開始使用 (執行方法)
本應用程式已經打包為獨立的可執行檔。您只需要：
1. 進入專案內的 **`dist/`** 資料夾。
2. 雙擊執行 **`SmartCalendar.exe`**。
3. 程式會自動讀取同目錄下的設定檔 (`todos.json`, `stock_config.json`, `weather_config.json`) 並開啟主視窗。
