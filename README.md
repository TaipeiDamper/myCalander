# Todo List 月曆應用程式

一個基於 Python Tkinter 的桌面 todo list 應用程式，包含互動月曆功能和任務管理。

## 版本資訊

- **當前版本**: v1.0.0
- **建立日期**: 2024-01-XX
- **Python 版本**: 3.7+

## 功能特色

### 核心功能

1. **互動月曆視圖**
   - 顯示當前月份的月曆
   - 標記有 todo 的日期（顯示任務標題）
   - 點擊日期查看該日的任務
   - 支援月份切換（上一個月/下一個月）
   - 今天的日期會特別標記

2. **Todo 列表視圖**
   - 顯示選中日期的所有任務
   - 顯示所有未完成的任務（當未選擇日期時）
   - 自動按時間排序（越接近完成時刻越上面）
   - 支援拖曳重新排序（基礎實作）

3. **任務管理**
   - 新增任務：標題、內容、日期、時間
   - 編輯任務：修改標題、內容、日期、時間
   - 刪除任務：確認後刪除
   - 任務標題會顯示在月曆上

4. **資料儲存**
   - 使用 JSON 檔案儲存（`todos.json`）
   - 自動載入和儲存
   - 關閉程式時自動儲存

## 安裝與執行

### 系統需求

- Python 3.7 或更高版本
- Tkinter（通常已內建於 Python）

### 執行方式

1. 確保已安裝 Python 3.7+
2. 在專案目錄下執行：

```bash
python main.py
```

### 檢查 Tkinter 是否可用

如果遇到 Tkinter 相關錯誤，可以執行以下命令檢查：

```bash
python -m tkinter
```

如果出現錯誤，請根據您的作業系統安裝 Tkinter：
- **Windows**: 通常已內建
- **Linux**: `sudo apt-get install python3-tk`
- **macOS**: 通常已內建

## 使用說明

### 基本操作

1. **查看月曆**
   - 程式啟動後會顯示當前月份的月曆
   - 有任務的日期會顯示任務標題
   - 使用 ◀ 和 ▶ 按鈕切換月份

2. **查看任務**
   - 點擊月曆上的日期，查看該日的任務
   - 或點擊「所有待完成任務」查看所有未完成的任務

3. **新增任務**
   - 在任務列表視圖中，點擊「+ 新增任務」按鈕
   - 填寫標題（必填，會顯示在月曆上）、內容、日期、時間
   - 點擊「確定」儲存

4. **編輯任務**
   - 在任務列表中，雙擊任務或點擊「編輯」按鈕
   - 修改任務資訊後點擊「確定」

5. **刪除任務**
   - 在任務列表中，點擊「刪除」按鈕
   - 確認後刪除

6. **排序任務**
   - 任務會自動按時間排序（越接近完成時刻越上面）
   - 可以透過拖曳手柄（☰）調整順序（基礎功能）

### 資料格式

任務資料儲存在 `todos.json` 檔案中，格式如下：

```json
{
  "todos": [
    {
      "id": "todo_20240115143000",
      "title": "任務標題",
      "content": "任務詳細內容",
      "date": "2024-01-15",
      "time": "14:30",
      "completed": false,
      "order": 0
    }
  ]
}
```

## 架構說明

### 設計模式

採用簡化版的 MVC（Model-View-Controller）架構：

- **Model**: `models.py` - Todo 資料模型
- **View**: `calendar_view.py`, `todo_list_view.py`, `todo_editor.py` - 使用者介面元件
- **Controller**: `main.py` - 主程式，協調各模組

### 資料流程

```
使用者操作
    ↓
主程式 (main.py)
    ↓
視圖元件 (calendar_view / todo_list_view)
    ↓
資料模型 (models.py)
    ↓
資料管理 (data_manager.py)
    ↓
JSON 檔案 (todos.json)
```
