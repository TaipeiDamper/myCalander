# 🎉 智能曆 (SmartCalendar) 開發日誌

這份日誌負責記錄目前專案所具備的功能，以及未來能考慮擴展或新增的想法，確保開發有條理地進行。

## 📍 目前擁有的功能 (現有功能)

### 1. 任務與月曆核心 (Calendar & Todo)
- **互動式月曆視圖**：包含動態計算的月份、日期轉換，以及標記任務的功能。如果當日有任務，會以不同顏色提示。
- **任務管理清單**：點擊特定日期可進入清單模式，支援新增、修改、刪除待辦事項，並清楚區分「已完成」與「未完成」。
- **資料持久化**：所有的任務均存在 `todos.json` 之中，以利離線狀態下的使用。

### 2. 左側磁吸側邊欄 (Sidebar)
- **獨立懸浮視窗設計**：側邊欄沒有系統邊框，隨時釘選在主視窗的特定側邊。
- **智慧同步與磁吸邏輯**：當主視窗移動、縮小、還原，側邊欄會緊密跟隨，並在觸發相關小工具時自動浮現。
- **模組化設計**：可任意加入 `register_widget` 開啟自定義的工具塊。

### 3. 多功能生活小工具 (Widgets)
- **股票追蹤板 (`stock_widget.py`)**：定時或手動更新自選清單內的台灣股市資訊，並且結合檔案日誌記錄歷史資料。
- **天氣資訊板 (`weather_widget.py`)**：顯示當地的天氣資訊，支援自訂不同的地理位置（經緯度、名稱），隨時提供室外環境與氣候速報。
- **番茄鐘與時鐘 (`clock_widget.py`)**：為辦公或專注模式提供番茄鐘 (Pomodoro Timer) 及即時時刻顯示。

### 4. 編譯與發布 (Build / Releasing)
- **自動化 exe 打包**：撰寫 `build_exe.py` 可將應用程式自動透過 PyInstaller 打包成獨立的 Windows 執行檔 (`.exe`)，並連同需要的 `json` 設定檔案一起部署在 `dist/` 目錄中。

## 🚀 未來能考慮新增的擴充功能 (規劃中)

1. **主題與色彩系統 (Theming System)**
   - 加入深色模式 (Dark Mode) 或主題自訂功能，讓視覺風格更百變且一致。
   - 增加字體大小調整選項。

2. **跨裝置雲端同步 (Cloud Sync)**
   - 結合 Google Drive、Dropbox 或是 Firebase 將 `todos.json` / 天氣股票設定進行備份與同步。

3. **桌面小工具 (Desktop Overlay mode)**
   - 讓應用成為透明的常駐 Widgets 直接固定在 Windows 桌面底層。

4. **提醒通知 (Notification)**
   - 針對待辦任務，若標註特定時間，到達前利用 Windows 系統通知（Toast notifications）或聲音提醒使用者。

5. **數據統整儀表板 (Dashboard / Weekly Review)**
   - 自動結算使用者一週之內完成了幾次番茄鐘、哪些時候任務積壓最多，產出一個簡單的回顧圖表。

6. **股市或天氣警報**
   - 如果下雨機率超過百分比，或是追蹤股票觸及設定價格，在系統內跳出小提醒彈窗。

## 📂 專案架構概觀
為達到高模組化及整潔的目的，專案檔案已針對領域功能重新分類：
- `core/`: 存放基底資料模型、側邊欄控制與基礎時鐘元件。
- `todo/`: 專注於月曆、檢視列表與任務編輯。
- `weather/`: 天氣小工具及專屬天氣配置。
- `stock/`: 股票追蹤小工具及股票相關除錯指令檔。
- 主目錄：僅保留開發用的 `main.py`（主程式入口）、`DEV_LOG.md`（日誌）、`build_exe.py` 和基礎的環境配置檔案。
